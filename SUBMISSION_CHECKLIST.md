# Submission Checklist (Part 1)

## Repo
- GitHub repo is public or shared with assessor access.
- Commits show progress across days (docs + prototype + refinements).

## Required Documentation
- `high-concept.md`
- `plan.md`
- `requirements.txt`
- `refinements-changes.md`
- `README.md` includes:
  - run instructions
  - credits/AI usage
  - AI Ethics Statement
  - reflections (200 words each)

## Videos (MP4)
- Day 2 demo (2–5 min)
- Day 3 demo (2–5 min)
- Each video narrates:
  - core loop
  - AI contributions + manual changes
  - at least 1 stability/refinement improvement

## Build / Zip
- Zip the full project folder (the repo root).
- Verify a fresh clone can run with:
  - `pip install -r requirements.txt`
  - `python -m src.main`
